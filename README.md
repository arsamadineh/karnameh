# کارنامه (KarNama)

سامانه هوشمند، بومی و مینیمال مدیریت پروژه‌ها، کارها و چک‌لیست‌های روزانه. طراحی‌شده با رعایت استانداردهای تجربه کاربری مدرن، تعاملات روان و پشتیبانی کامل از چینش راست‌به‌چپ (RTL) و زبان فارسی.

## قابلیت‌های کلیدی

- **داشبورد مرکزی**: نمایش وضعیت کلی پروژه‌ها، کارهای در دست اقدام و پیشرفت چک‌لیست‌های روزانه به صورت خلاصه و آماری.
- **مدیریت مشتریان**: سازماندهی اطلاعات تماس مشتریان، رنگ اختصاصی برای هر مشتری و مشاهده بودجه کل پروژه‌های مرتبط با هر مشتری به صورت آنی.
- **کنترل پروژه‌ها**: ثبت و پیگیری وضعیت پروژه‌ها (در انتظار، در حال انجام، انجام شده) به همراه اولویت‌بندی هوشمند و اتصال مستقیم به مشتریان مرتبط.
- **سامانه وظایف (بورد کانبان)**: سیستم سازماندهی کارها به روش کشیدن و رها کردن با طبقه‌بندی وضعیت‌ها و زمان‌بندی دقیق.
- **چک‌لیست روزانه**: ایجاد لیست کارهای روزانه به همراه نوار پیشرفت درصد پیشروی و ناوبری زمانی آسان بین روزها.
- **سیستم چندزبانه و بومی**: طراحی هماهنگ با تقویم خورشیدی و اعداد فارسی.
- **موتور حرکتی روان**: انیمیشن‌های بهینه‌سازی‌شده برای نرخ نوسازی ۱۲۰ هرتز شبیه به نرم‌افزارهای بومی سیستم‌عامل macOS.

## پیش‌نیازهای توسعه محلی

برای اجرای پروژه در محیط توسعه محلی به ابزارهای زیر نیاز دارید:

- **Node.js** نسخه LTS یا بالاتر
- **Rust** و ابزارهای کامپایل مرتبط با سیستم‌عامل شما

## راه‌اندازی و اجرای محلی

۱. ابتدا وابستگی‌های پروژه را نصب کنید:

```bash
npm install
```

۲. پروژه را در حالت توسعه اجرا کنید (اجرای دسکتاپ تائوری):

```bash
npm run tauri dev
```

۳. برای ساختن فایل نصبی نهایی سیستم‌عامل خود به صورت محلی:

```bash
npm run tauri build
```

## معماری و ساختار پروژه

پروژه کارنامه با معماری مدرن و تفکیک کامل لایه‌ها طراحی شده است:

- **پوسته کاربری (Frontend)**: توسعه داده شده با SolidJS و TypeScript برای رندرینگ سریع و بهینه‌سازی مصرف رم.
- **هسته پردازشی (Backend)**: Rust و Tauri نسخه ۲ جهت تعامل امن با منابع سیستم و پایگاه‌داده محلی SQLite.
- **طراحی بصری**: استفاده از کدهای CSS بومی (CSS Variables) برای حفظ انعطاف و سرعت بارگذاری زیر ۱ ثانیه.
